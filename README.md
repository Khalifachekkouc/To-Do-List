# 📝To-Do List App

A modern, responsive To-Do List application built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project features a unique "notebook" aesthetic using Google Fonts and a dotted paper background, designed to help users stay organized with a tactile feel.

## 🚀 Features

* **Full CRUD Functionality**: Create, Read, Update, and Delete tasks easily.
* **Task Categorization**: Organize tasks by custom categories (Work, Home, Study, etc.).
* **Priority System**: Assign High, Medium, or Low priority to tasks with color-coded badges.
* **Real-time Search**: Instant filtering of tasks by title or category as you type.
* **Local Storage**: Your data is saved locally in your browser, so your tasks persist even after a page refresh.
* **Responsive Design**: Optimized for both desktop and mobile devices.
* **Status Toggle**: Mark tasks as "Done" or "Undo" with a single click.

## 🛠️ Tech Stack

* **HTML5**: Semantic structure.
* **CSS3**: Custom layouts, Grid/Flexbox, and "Notebook" styling.
* **JavaScript (ES6+)**: DOM manipulation, LocalStorage API, and filtering logic.
* **Google Fonts**: 'Architects Daughter' and 'Edu TAS Beginner' for the handwritten look.
